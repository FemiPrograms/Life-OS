# Life OS

A web app that helps students track placement, internship, and graduate job applications in one place.

## What it does

Users can:

* add and track job applications
* manage deadlines and interview stages
* record company notes
* track required skills
* review weekly progress
* later use AI to analyse job descriptions and suggest next steps

## Why I built it

Students often manage applications across spreadsheets, emails, notes, and browser tabs. This project creates one organised system to make the process clearer and more strategic.

## Planned features

* Application tracker
* Deadline dashboard
* Skill gap tracker
* Weekly review system
* AI job description analyser
* CV/document upload
* Cloud deployment

Tech stack

* Frontend: React / Next.js
* Backend: Python FastAPI
* Database: PostgreSQL
* Cloud: AWS
* Storage: Amazon S3
* Testing: Pytest
* CI/CD: GitHub Actions
